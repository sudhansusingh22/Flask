Github OAuth
=========
### Objective :
* Authenticate with Github using Github OAuth and Python Flask.
* Fetch the repositories of a particular use.
* Display those repositories in the Dashboard.


### Dependencies
------------
* Flask
* Bootstrap

### Steps To Run:
-------
```
python run.py
```
